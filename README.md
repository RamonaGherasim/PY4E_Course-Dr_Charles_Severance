# PY4E Course || Author: Professor Charles Serverance



##### In this repository you can find my solved exercises and some worked examples of the Python 4 Everybody Course taught by Professor [Charles Serverance](https://github.com/csev).
##### This free course can be found [**_here_**](https://www.py4e.com/).
##### This is an introductory course that can be taken even if have no programming background.
